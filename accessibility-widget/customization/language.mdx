## Overview

Select the language you want to display the widget in. The widget's labels, buttons, and tooltips use the selected language.

## Language dropdown

Use the **Widget language** dropdown to choose a language. For example, **English (English)** displays the widget interface in English. The list includes all supported languages for the widget.

<Tip>
Choose a language that matches your store's primary audience. The widget language is independent of your Shopify store language.
</Tip>
